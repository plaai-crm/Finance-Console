# คู่มือเปิดใช้ “ตั้งค่าแจ้งเตือนเอง” (สำหรับคนไม่เขียนโค้ด)

ตอนนี้คุณเลือกได้เองแล้วว่าจะให้ระบบเตือน **ล่วงหน้ากี่วัน** และ **พิมพ์ข้อความแจ้งเตือนเอง** ได้ ทั้งในโปรแกรมและใน LINE โดยตั้งค่าจากหน้า “ตั้งค่า” ในแอปโดยตรง ไม่ต้องแตะโค้ดอีกเลย

มีของให้ 3 ไฟล์:

1. **index.html** — โปรแกรมเวอร์ชันใหม่ (มีหน้าตั้งค่าแจ้งเตือนแล้ว)
2. **line_notify_v2.sql** — ไฟล์อัปเกรดฝั่ง LINE (รันครั้งเดียว)
3. คู่มือฉบับนี้

---

## ทำ 2 ขั้นตอนนี้ครั้งเดียว แล้วใช้ได้ตลอด

### ขั้นที่ 1 — เอาโปรแกรมใหม่ไปแทนของเดิม
เอาไฟล์ **index.html** ใหม่ ไปวางแทนไฟล์เดิมที่คุณใช้อยู่ (ที่เดียวกับที่เปิดโปรแกรมทุกวัน) แค่นี้หน้าตั้งค่าก็พร้อมแล้ว

> ถ้าใช้แบบเปิดในเครื่องเฉย ๆ (ยังไม่ต่อ LINE) ก็ตั้งค่า “จำนวนวัน” และ “ข้อความ” ใช้ในโปรแกรมได้เลย ข้ามขั้นที่ 2 ไปก่อนได้

### ขั้นที่ 2 — อัปเกรดฝั่ง LINE (ให้ค่าที่ตั้งในแอปส่งเข้า LINE ด้วย)
> ทำต่อจากที่คุณเคยรัน `line_notify.sql` (ตัวใส่โทเคน LINE) ไปแล้ว — ไฟล์ใหม่นี้ **ไม่ลบโทเคนเดิม** ของคุณ

1. เข้าเว็บ **Supabase** → เข้าโปรเจกต์ของคุณ
2. เมนูซ้าย เลือก **SQL Editor** → กด **New query**
3. เปิดไฟล์ **line_notify_v2.sql** คัดลอก **ทั้งหมด** มาวางในช่อง
4. กดปุ่ม **Run** (เขียว มุมขวาล่าง) → รอขึ้นว่า Success
5. เสร็จแล้ว ปิดได้เลย

---

## วิธีใช้งานประจำวัน (ในแอป)

เข้าโปรแกรม → เมนู **ตั้งค่า** → การ์ด **“แจ้งเตือนรายจ่าย (ในโปรแกรม + LINE)”**

- **แจ้งล่วงหน้ากี่วันก่อนครบกำหนด** — พิมพ์ตัวเลข เช่น `3`, `7`, `14`
  (ใส่ `0` = เตือนเฉพาะที่ครบกำหนดวันนี้หรือเลยกำหนด)
- **ข้อความแจ้งเตือน (พิมพ์เอง)** — พิมพ์ข้อความที่อยากให้ขึ้นหัวข้อความ เช่น
  *“อย่าลืมโอนจ่ายบิลด้วยนะคะ 😊”* — เว้นว่างไว้ก็ได้ ระบบจะใช้ข้อความมาตรฐาน
- **ตัวเลือกการส่ง LINE**
  - *ส่งทุกวันแม้ไม่มีรายการค้าง* / *ส่งเฉพาะเมื่อมีรายการค้าง*
  - *เปิด / ปิด แจ้งเตือน LINE*
- กด **บันทึกการตั้งค่า** → ค่าจะมีผลทันทีทั้งในแอปและ LINE
- ปุ่ม **ส่งทดสอบเข้า LINE** — กดเพื่อลองส่งเข้า LINE ดูเดี๋ยวนั้น (ส่งได้แม้ยังไม่มีรายการค้าง)

ระบบจะส่งสรุปรายจ่ายเข้า LINE ให้อัตโนมัติ **ทุกเช้า 07:00 น.** ตามจำนวนวันและข้อความที่คุณตั้งไว้

---

## เช็คว่าใช้ได้ไหม
1. ในแอป ตั้งจำนวนวัน + พิมพ์ข้อความ → กด **บันทึกการตั้งค่า** (ต้องขึ้น “บันทึกการตั้งค่าแล้ว”)
2. กด **ส่งทดสอบเข้า LINE** → ควรเด้งเข้า LINE ภายในไม่กี่วินาที โดยหัวข้อความเป็นข้อความที่คุณพิมพ์

ถ้าปุ่มทดสอบไม่เข้า LINE ให้ตรวจว่าได้รัน `line_notify_v2.sql` (ขั้นที่ 2) แล้ว และก่อนหน้านี้ตั้งค่าโทเคน LINE ใน `line_notify.sql` ครบ

---

## สรุปสั้น ๆ
เอา index.html ไปแทนของเดิม → รัน line_notify_v2.sql บน Supabase หนึ่งครั้ง → จากนั้นปรับวัน/ข้อความได้เองจากหน้า “ตั้งค่า” ในแอปตลอดไป ✅
